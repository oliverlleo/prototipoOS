Como Usar Esta Página

Esta página exibe formulários para as entidades que você criou no construtor. Para visualizar um formulário específico, adicione o caminho dele na URL.

Exemplo:

.../user-view.html?path=vendas/cliente

Onde "vendas" é o nome do módulo e "cliente" é a entidade.
